前端技術


學習前端需了解各項技術原理的運作，特別重的是在 「觀念建立」、「反覆練習」、「互相討論」 的循環，只有不斷嘗試才能在過程中找出自己常犯的錯誤，為自己建立厚實的基礎。 日後在期末專題才能發揮所學，設計出結構穩固、兼具美觀、互動功能的網站。 若有遇到不了解或是忘記的語法，可以擅用網路資源，例如： Google 、 MDN 或 W3school 查詢。

HTML-負責架構、內容

超文本標記內容（英文：HyperText Markup language，簡稱：HTML）是一種用於建立網頁的標準標記語言。 HTML 是一種基礎技術，常與CSS 、JavaScript 一起被眾多網站用於設計賞心悅目的網頁、網頁用程式以及行動應用程式的使用者介面。
